# TravelToursWebsite Admin Dashboard Docs

This folder is the persistent source of truth for completing the React admin dashboard in `src-admin-dashboard`.

When resuming in a new Codex context:

1. Read `admin-dashboard-docs/admin-dashboard-plan.md`.
2. Read `admin-dashboard-docs/admin-dashboard-tracker.md`.
3. Continue from the first phase whose status is not `Done`.
4. After completing a phase, update `admin-dashboard-docs/admin-dashboard-tracker.md` before stopping.

Important operating rules:

- Codex must not run Git commands for this dashboard repo.
- The user handles branch creation, commits, pushes, and pull requests.
- Do not start the next phase until the user has reviewed the completed phase and asks to continue.
- Keep implementation work inside `src-admin-dashboard` unless a dashboard workflow is impossible without an API change.
